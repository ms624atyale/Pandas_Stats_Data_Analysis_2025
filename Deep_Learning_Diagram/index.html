Step 1: Input to Hidden Representation

Instead of just $h = Wx$, we must add the unique node biases (+ b₁) and then squeeze the output through a non-linear ReLU activation function:

h = ReLU( W₁·x + b₁ )

Expanded Vector Form:

[h₁, h₂, h₃]ᵀ = ReLU( [W₁₁..W₃₂]·[x₁, x₂]ᵀ + [b₁, b₂, b₃]ᵀ )

Where z = W₁·x + b₁ represents the linear aggregation (pre-activation), and the Activation function turns negative values into 0: h = max(0, z).

Step 2: Hidden to Output Prediction

The processed hidden attributes h are fed into the second transformation layer to generate predictions:

ŷ = W₂·h + b₂

Expanded Vector Form:

[ŷ₁, ŷ₂]ᵀ = [W'₁₁..W'₂₃]·[h₁, h₂, h₃]ᵀ + [b'₁, b'₂]ᵀ

Each target output receives its own distinct bias adjustment factor to shift the final prediction scale correctly.
