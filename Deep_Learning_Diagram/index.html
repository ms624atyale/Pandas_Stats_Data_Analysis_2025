1. Input to Hidden Transition

Each hidden neuron first aggregates the weighted inputs, adds its corresponding bias term (b₁), and then passes the result through the ReLU activation function:

h = ReLU( W₁·x + b₁ )

Fully Expanded Matrix Form:

┌ h₁ ┐         ┌ ┌ w₁₁ w₁₂ ┐ ┌ x₁ ┐   ┌ b₁₁ ┐ ┐
│ h₂ │ = ReLU │ │ w₂₁ w₂₂ │ │ x₂ │ + │ b₁₂ │ │
└ h₃ ┘         └ └ w₃₁ w₃₂ ┘ └ ┘   └ b₁₃ ┘ ┘

*Where ReLU(z) = max(0, z). If the sum is negative, it gets clipped to 0.

2. Hidden to Output Transition

The processed activation states h are mapped to the final predictions ŷ using the second weight matrix and the output layer biases (b₂):

ŷ = W₂·h + b₂

┌ ŷ₁ ┐    ┌ w'₁₁ w'₁₂ w'₁₃ ┐ ┌ h₁ ┐   ┌ b'₁₁ ┐
└ ŷ₂ ┘ = │ w'₂₁ w'₂₂ w'₂₃ │ │ h₂ │ + │ b'₁₂ │
         └ ┘ │ h₃ │   └ ┘
