How this connects to Matrix Operations & Loss

Step 1: The Input Matrix x

Our input layer gathers height and weight. We pack this physical data into a vector:

x = [170, 70]ᵀ

This inputs into a 2-node column vector, feeding forward into our calculations.

Step 2: Hidden Transformations

We transform inputs into abstract combinations using a matrix multiplication + shift (Bias) and filter out negative values using the ReLU Activation Function:

h = max(0, W₁x + b₁)

These hidden nodes extract high-level health patterns that raw height and weight can't express directly.

Step 3: Calculating and Using Loss

The output layer predicts Blood Pressure and Triglycerides ($\hat{y}$). We then calculate error against True Values ($y$):

Loss = ½ [(y₁ - ŷ₁)² + (y₂ - ŷ₂)²]

In deep learning, learning is mathematically pushing this Loss value towards zero. We do this by passing the error signal backwards (feedback) and slightly adjusting all 17 parameters.
