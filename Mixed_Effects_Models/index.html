Understanding a Simple Mixed-Effects Model

A beginner-friendly example using speech rate (fast vs. slow) and C type (fortis, aspirated, lenis) to predict the F0 of the following vowel (V), while treating 30 subjects as a random effect.

1. What are we trying to explain?

In this example, the outcome we want to predict is the F0 of the vowel after a consonant. F0 is a measure related to pitch.

Dependent variable: F0 of V
Unit: Hz (illustrative)

We want to know whether vowel F0 changes depending on:

  • Speech rate: fast vs. slow
  • C type: fortis, aspirated, lenis

2. Fixed effects vs. random effects

Fixed effects are the factors we want to study directly.

  • Speech rate (fast / slow)
  • C type (fortis / aspirated / lenis)

Random effects are sources of variation we want to control for, not the main focus.

  • Subject (30 people)

Why include subjects as a random effect? Because some people naturally produce higher or lower F0 than others. A mixed-effects model allows each subject to have their own baseline.

3. Start from basic linear regression

A mixed-effects model is an extension of the familiar regression idea:

y = ax + b
  • y = outcome we want to predict (here, F0 of V)
  • x = predictor (for example, speech rate)
  • a = slope (how much y changes when x changes)
  • b = intercept (starting point)

In real speech data, we usually have more than one predictor, and we often need to account for repeated measurements from the same speakers. That is why we use a mixed-effects model.

4. The mixed-effects model in this example

Here is a simple model formula:

F0_of_V ~ speech_rate + C_type + (1 | subject)

This means:

  • F0_of_V is predicted by speech_rate and C_type.
  • (1 | subject) means each of the 30 subjects can have a different baseline F0.
In plain English: “We test whether speech rate and consonant type affect vowel F0, while controlling for the fact that the same subjects may have different average pitch levels.”

6. How to read the plot

  • If the dots at x = 1 (fast) tend to be higher than the dots at x = 0 (slow), fast speech may be associated with higher F0.
  • If the three colored groups sit at different heights, that suggests an effect of C type.
  • The dashed line shows the overall direction of the relationship for each consonant type.

7. Example interpretation

Suppose the model finds:

  • a positive effect of fast speech rate → vowel F0 is higher in fast speech than in slow speech
  • a difference between lenis and fortis → vowel F0 differs by consonant type

Then we could say:

“Vowel F0 was affected by speech rate and consonant type. In addition, the model controlled for subject-to-subject differences by including subject as a random intercept.”

8. Why not use ordinary regression only?

Ordinary regression assumes all observations are independent. But speech experiments often collect many observations from the same speakers. Those observations are not fully independent, because each speaker has their own speaking habits.

A mixed-effects model solves this by allowing the model to include:

  • fixed effects: the linguistic factors we care about
  • random effects: speaker-to-speaker variation (and sometimes item-to-item variation too)

9. Take-home message

  • Dependent variable: F0 of the vowel
  • Fixed effects: speech rate and C type
  • Random effect: subject (30 speakers)
  • Regression idea: start from y = ax + b, then extend it to multiple predictors and random effects
One-sentence summary: A mixed-effects model is like regression, but smarter for repeated speech data because it can test linguistic effects while also controlling for differences across speakers.
